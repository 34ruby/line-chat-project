# line-chat-project
LINEメッセージングAPIを利用したウェブ相談対応サービス

## Install the dependencies
npm install

## Start the app in development mode
client
```bash
npm run dev
```
server
```bash
cd server
node index.js
```

## List of Technologies Used
vue.js, vuex, quasar ui, scss<br>
node.js, express.js, mongodb, socket.io, axios, moment.js<br>
LINE messaging API
